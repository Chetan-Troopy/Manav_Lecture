20 Practical Tasks Using Map, Filter, and Reduce

🔁 Map Tasks

  • Convert Celsius to Fahrenheit
    Given an array of Celsius temperatures, use map to convert them to Fahrenheit.
  • Capitalize First Letter
    Convert an array of lowercase strings to capitalized format (e.g., ["apple"] → ["Apple"]).
  • Extract Usernames
    Given an array of user objects, extract only the usernames into a new array.
  • Double the Numbers
    Given an array of numbers, return a new array with all numbers doubled.
  • Append Index
    Add the index position to each string (e.g., ["car", "bike"] → ["car0", "bike1"]).

🚫 Filter Tasks

  • Filter Even Numbers
    Return only the even numbers from an array.
  • Filter Adults
    Given an array of objects with age, filter out people who are 18 or older.
  • Remove Falsy Values
    Filter out all falsy values (false, 0, null, undefined, NaN, "") from an array.
  • Words Longer Than 5
    Given an array of strings, return words with length > 5.
  • Filter Prime Numbers
    From a given array, return only the prime numbers.

➕ Reduce Tasks

  • Sum of All Numbers
    Return the sum of all numbers in an array.
  • Find Maximum Number
    Use reduce to find the maximum value in a numeric array.
  • Flatten an Array
    Flatten a nested array like [[1, 2], [3, 4]] → [1, 2, 3, 4].
  • Count Occurrences
    Count how many times each element appears in an array.
  • Concatenate Strings
    Combine an array of strings into a single sentence.

🔁🧠 Combination Tasks (Map + Filter + Reduce)

  • Square and Filter Odds
    Square each number using map, then use filter to keep only odd squares.
  • Total Price with Tax
    Given an array of product prices, add 18% tax using map, then reduce to find the total.
  • Unique Values
    Use reduce to create an array of unique values from a list (without using Set).
  • Find Average Age
    Given a list of user objects with age, calculate the average age.
  • Word Frequency Counter
    Given a string, split into words, and use reduce to count occurrences of each word.
