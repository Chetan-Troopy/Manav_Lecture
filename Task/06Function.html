20 Practical Tasks on JavaScript Functions

  1. Create a normal function greet() that takes a name and returns "Hello, [name]!".
  2. Write a function multiply(a, b = 2) that multiplies two numbers with a default parameter.
  3. Create a function add(a, b) that returns the sum of two numbers.
  4. Write a function isEven(num) that checks if a number is even or odd.
  5. Create a function findMax(a, b) to find the larger number between two inputs.
  6. Write an arrow function square(num) to return the square of a number.
  7. Create an arrow function subtract(a, b) to subtract two numbers.
  8. Write an arrow function createUser that returns an object with properties name and age.
  9. Create a function expression divide(a, b) that divides two numbers.
  10. Write a function expression isPrime(num) to check if a number is prime.
  11. Create a function expression factorial(n) to compute the factorial of a number.
  12. Write an immediately invoked function expression (IIFE) that logs "I am an IIFE".
  13. Create a function createMultiplier(factor) that returns a function to multiply a number by the given factor.
  14. Write a function expression reverseString(str) that reverses a string.
  15. Create an arrow function findMin(arr) to find the smallest number in an array.
  16. Write a function expression average(arr) to calculate the average of numbers in an array.
