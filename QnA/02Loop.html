JavaScript Loops - Q&A

1. What are the different types of loops in JavaScript?

JavaScript provides for, while, do-while, for-in, and for-of loops.

2. How does a for loop work?

A for loop executes a block of code repeatedly with a counter.

for (let i = 0; i < 5; i++) { console.log(i); }
3. What is the syntax of a while loop?
while (condition) { /* Code */ }
4. How is a do-while loop different from a while loop?

A do-while loop executes at least once, even if the condition is false.

5. What happens if the condition in a while loop is always true?

It creates an infinite loop, which can crash the program.

6. How do you use a for-in loop in JavaScript?

It loops over the properties of an object.

7. How does a for-of loop work?

It iterates over elements of an iterable (e.g., arrays).

8. What is the purpose of the break statement in loops?

It exits the loop immediately.

9. How does the continue statement affect loop execution?

It skips the current iteration and proceeds to the next one.

10. What is an infinite loop, and how can you prevent it?

A loop without an exit condition. Always ensure the loop eventually ends.

11. How can you loop through an array using a for loop?
for (let i = 0; i < arr.length; i++) { console.log(arr[i]); }

13. What is the difference between for and forEach()?

forEach() is a method for arrays and cannot be stopped with break.

14. How can you use a loop to generate numbers from 1 to 100?
for (let i = 1; i <= 100; i++) { console.log(i); }
15. How do you exit a loop when a certain condition is met?

Using break.

16. How can you loop through a string character by character?

Using a for loop.

17. What is the purpose of using nested loops?

For multi-dimensional operations, such as matrix traversal.

18. How can you reverse an array using loops?

Using a for loop to swap elements.

19. What are the benefits of using loops in JavaScript?

They reduce redundant code and automate repetitive tasks.

20. How do you create a loop that counts down from 10 to 1?
for (let i = 10; i >= 1; i--) { console.log(i); }
