Conditional Statements in JavaScript - Q&A

1. What are the different types of conditional statements in JavaScript?

JavaScript provides if, if-else, if-else if-else, switch, and the ternary operator (? :) for decision-making.

2. How does an if statement work in JavaScript?

An if statement executes a block of code if the given condition evaluates to true.

if (x > 10) {
    console.log("X is greater than 10");
}

3. What is the difference between if and else if?

if executes the first condition that evaluates to true, whereas else if allows checking multiple conditions before the final else executes.

4. Can you use multiple else if statements in a condition?

Yes, you can use multiple else if conditions before the final else.

if (score >= 90) {
    console.log("Grade A");
} else if (score >= 80) {
    console.log("Grade B");
} else {
    console.log("Grade C");
}

5. What happens if none of the conditions in an if-else if-else block are met?

If there's no else statement, nothing happens. If else is provided, its block executes.

6. How does the switch statement work in JavaScript?

The switch statement checks a variable against multiple case values and executes the corresponding code.

let day = 3;
switch (day) {
    case 1: console.log("Monday"); break;
    case 2: console.log("Tuesday"); break;
    default: console.log("Other day");
}

7. What is the purpose of the default case in a switch statement?

The default case runs when no case matches the given value.

8. How can you combine multiple cases in a switch statement?

Multiple cases can share the same code block.

switch (fruit) {
    case "apple":
    case "banana":
        console.log("This is a fruit.");
        break;
}

9. What is the difference between switch and if-else statements?

if-else is better for complex conditions, whereas switch is useful when checking a single variable against multiple values.

10. How do you use logical operators (&&, ||, !) inside an if condition?

Logical operators help combine multiple conditions.

if (age >= 18 && hasID) {
    console.log("Allowed");
}

11. What happens if a condition inside an if statement evaluates to false?

The code inside the if block is skipped.

12. Can you use a ternary operator (? :) instead of an if-else statement?

Yes, it is a shorthand for simple conditions.

let result = (age >= 18) ? "Adult" : "Minor";

13. How do you check for multiple conditions using && and ||?

&& requires all conditions to be true, while || requires at least one.

if (x > 0 && y > 0) { console.log("Both positive"); }
if (x > 0 || y > 0) { console.log("At least one positive"); }

14. What is the difference between == and === in conditional statements?

== checks for value equality, while === checks for value and type equality.

15. How do falsy values affect conditional statements in JavaScript?

Falsy values (0, "", null, undefined, NaN, false) evaluate as false in conditions.
