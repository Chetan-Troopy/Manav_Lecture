Classes In JavaScript

Classes are a template for creating objects. They encapsulate data with code to work on that data. Classes in JS are built on prototypes but also have some syntax and semantics that are unique to classes.

Objects are Instance of Class

Key Features of JavaScript Classes

  1. Constructor Method
    • Runs when an Object is Created
    • Used to Initialize Properties
      • 
            
                constructor(name, age) { 
                    this.name = name; 
                    this.age = age; 
                }
  2. Instance Method
    • Defined Inside the class and available to all instances
      • 
            
                
                greet() {
                    console.log(`Hi, I'm ${this.name} and I'm ${this.age} year old`);
                }
  3. Inheritance using extend and super
    4. 
      
      
        class Student extends Person {
            constructor(name, age, course) {
              super(name, age); // Calls the parent constructor
              this.course = course;
            }
    
            study() {
              console.log(`${this.name} is studying ${this.course}`);
            }
          }
