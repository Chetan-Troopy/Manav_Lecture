Object-Oriented Programming in JavaScript

Learn the 4 pillars of OOP with practical examples in JavaScript

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes to structure code.

Here are the four pillars of OOP:

  • Encapsulation: Bundling data and methods that operate on that data
  • Abstraction: Hiding the complex details and showing only the necessary parts
  • Inheritance: Child classes inherit properties and methods from parent classes
  • Polymorphism: Methods behaving differently depending on the object

OOP in JavaScript - Examples

1. Basic Object Example


const person = {
    name: "Alice",
    age: 25,
    greet() {
        console.log(\`Hi, I'm \${this.name}\`);
    }
};

person.greet(); // Hi, I'm Alice

2. Function Constructor Example


function Person(name, age) {
    this.name = name;
    this.age = age;
}

Person.prototype.greet = function() {
    console.log(\`Hello, I am \${this.name}\`);
};

const p1 = new Person("Bob", 30);
p1.greet(); // Hello, I am Bob

3. ES6 Classes Example


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(\`Hi, I'm \${this.name}\`);
    }
}

const user = new Person("Charlie", 28);
user.greet(); // Hi, I'm Charlie

4. Inheritance Example


class Employee extends Person {
    constructor(name, age, position) {
        super(name, age);
        this.position = position;
    }

    work() {
        console.log(\`\${this.name} is working as \${this.position}\`);
    }
}

const emp = new Employee("Dana", 35, "Manager");
emp.greet(); // Hi, I'm Dana
emp.work();  // Dana is working as Manager

5. Polymorphism Example


class Animal {
    speak() {
        console.log("Animal speaks");
    }
}

class Dog extends Animal {
    speak() {
        console.log("Dog barks");
    }
}

const a = new Animal();
const d = new Dog();

a.speak(); // Animal speaks
d.speak(); // Dog barks
