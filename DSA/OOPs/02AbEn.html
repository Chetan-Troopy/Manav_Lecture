Object-Oriented Programming in JavaScript

Learn the 4 pillars of OOP with practical examples in JavaScript

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes to structure code.

Here are the four pillars of OOP:

  • Encapsulation: Bundling data and methods that operate on that data
  • Abstraction: Hiding the complex details and showing only the necessary parts
  • Inheritance: Child classes inherit properties and methods from parent classes
  • Polymorphism: Methods behaving differently depending on the object

Abstraction in JavaScript

Abstraction refers to hiding the complex details of an object and exposing only the essential parts.


class Car {
    constructor(model, engineStatus) {
        this.model = model;
        this.engineStatus = engineStatus;
    }

    startCar() {
        if (this.engineStatus === "off") {
            this.turnOnEngine();
        }
        console.log(\`\${this.model} is now running.\`);
    }

    turnOnEngine() {
        console.log("Engine starting...");
        this.engineStatus = "on";
    }
}

const myCar = new Car("Toyota", "off");
myCar.startCar(); // Toyota is now running.

Encapsulation in JavaScript

Encapsulation refers to bundling data and methods into a single unit (class) and restricting access to some of the object's components.


class BankAccount {
    #balance = 0;

    constructor(owner) {
        this.owner = owner;
    }

    deposit(amount) {
        if (amount > 0) {
            this.#balance += amount;
            console.log(\`Deposited \${amount}\`);
        } else {
            console.log("Invalid deposit amount.");
        }
    }

    withdraw(amount) {
        if (amount > 0 && this.#balance >= amount) {
            this.#balance -= amount;
            console.log(\`Withdrew \${amount}\`);
        } else {
            console.log("Invalid withdrawal amount or insufficient funds.");
        }
    }

    getBalance() {
        return this.#balance;
    }
}

const account = new BankAccount("John Doe");
account.deposit(500); // Deposited 500
account.withdraw(100); // Withdrew 100
console.log(account.getBalance()); // 400
