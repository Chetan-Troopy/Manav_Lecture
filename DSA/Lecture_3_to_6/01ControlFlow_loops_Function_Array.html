Control Flow, Loops, Functions, and Arrays

ðŸ“Œ 1. Control Flow in JavaScript

Control flow refers to the order in which individual statements, instructions, or function calls are executed. It allows you to control the path your program follows during its execution based on conditions.

âž¤ `if`, `else if`, `else` Statements

These statements are used for decision-making. Depending on the result of the condition, different blocks of code will execute.

if (condition) {
    // block of code
} else if (condition) {
    // block of code
} else {
    // block of code
}

Example:

let score = 85;

if (score >= 90) {
    console.log("Grade A");
} else if (score >= 75) {
    console.log("Grade B");
} else {
    console.log("Grade C");
}

âž¤ `switch` Statement

The `switch` statement is used when you have multiple conditions to check and is more readable for multiple `if` conditions.

switch (expression) {
  case value1:
    // block of code
    break;
  case value2:
    // block of code
    break;
  default:
    // block of code
}

Example:

let day = 2;

switch(day) {
    case 1: console.log("Monday"); break;
    case 2: console.log("Tuesday"); break;
    case 3: console.log("Wednesday"); break;
    default: console.log("Another day");
}

ðŸ“Œ 2. Loops in JavaScript

Loops allow you to repeat a block of code multiple times, which is useful when you need to perform the same operation on a collection of items or a range of numbers.

âž¤ `for` Loop

The `for` loop is commonly used when the number of iterations is known in advance.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Example:

for (let i = 0; i < 5; i++) {
    console.log(i); // prints 0, 1, 2, 3, 4
}

âž¤ `while` Loop

The `while` loop is useful when you want to repeat an action as long as a certain condition is true.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Example:

let i = 0;
while (i < 5) {
    console.log(i); // prints 0, 1, 2, 3, 4
    i++;
}

âž¤ `do-while` Loop

The `do-while` loop executes the code block first, then checks the condition after the loop. This guarantees that the loop runs at least once.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Example:

let i = 0;
do {
    console.log(i); // prints 0, 1, 2, 3, 4
    i++;
} while (i < 5);

ðŸ“Œ 3. Functions in JavaScript

Functions are blocks of reusable code that perform a specific task. You can call a function multiple times without repeating the code inside it.

âž¤ Function Declaration

Function declaration creates a named function that can be called by its name.

function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("Alice");

Example:

function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("Alice");  // Output: Hello, Alice!

âž¤ Function Expressions

A function expression assigns a function to a variable. This function can then be called using that variable.

const add = function(x, y) {
    return x + y;
};
console.log(add(2, 3)); // Output: 5

ðŸ“Œ 4. Arrays in JavaScript

Arrays are used to store multiple values in a single variable. They are useful when you need to work with lists of data, such as numbers, strings, or objects.

âž¤ Creating an Array

Arrays are created using square brackets `[]`.

let fruits = ["Apple", "Banana", "Cherry"];

Example:

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]

âž¤ Accessing Array Elements

Array elements are accessed by their index, which starts at 0.

console.log(fruits[0]); // Output: Apple

âž¤ Adding Elements to an Array

You can add elements to an array using the `push()` method, which adds items to the end of the array.

fruits.push("Orange");  // Adds "Orange" to the end
console.log(fruits); // Output: ["Apple", "Banana", "Cherry", "Orange"]

âž¤ Removing Elements from an Array

The `pop()` method removes the last element of the array.

fruits.pop();  // Removes "Orange"
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]
